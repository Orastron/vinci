# Vinci - Cross-Platform Windowing Library

Refer to `vinci.h` and `test.c` to get started with the API.

## Supported Platforms
- Linux   (via XCB)
- Windows (Win32)
- macOS   (Cocoa)

## Status
- Unstable
- Undocumented
- API might change

## Building
Run `make` to build the test program.
